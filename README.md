# Export-Finance-Analytics-Dashboard
Power BI dashboard analyzing 52,122 export finance deals from 2007-2025
## Project Overview
An interactive Power BI dashboard analyzing 52,122 export finance 
deals from 2007–2025 across Insurance, Working Capital, Guarantee, 
and Loan programs.

## Dashboard Features
- Authorization Trend 2007–2025 (Line Chart)
- Authorized Amount by Program (Bar Chart)
- Top 10 Countries by Authorization (Bar Chart)
- Program Performance Heatmap by Year
- Interactive slicers: Fiscal Year, Program, Country, Policy Type

## Key Metrics
- Total Deals: 52,122
- Programs: Insurance, Working Capital, Guarantee, Loan
- Countries: 100+
- Time Period: FY 2007–2025

## Technology Stack
- Microsoft Power BI Desktop
- Microsoft Excel (Data Cleaning)
- DAX (Data Analysis Expressions)
